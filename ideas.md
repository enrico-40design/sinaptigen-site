# Design Brainstorm - Synaptigen24 Presell Page

## Conceito Escolhido: Minimalismo Científico com Ênfase Emocional

### Design Movement
**Neuro-Minimalism**: Uma fusão entre o minimalismo moderno e a linguagem visual da neurociência, criando uma estética limpa mas com elementos que sugerem sofisticação científica.

### Core Principles
1. **Clareza Radical**: Cada elemento serve um propósito persuasivo. Sem decoração gratuita.
2. **Hierarquia Emocional**: O design guia o leitor através de uma jornada emocional (problema → esperança → ação).
3. **Autoridade Científica Acessível**: Usa elementos visuais que sugerem pesquisa e confiabilidade sem ser intimidador.
4. **Movimento Sutil**: Animações que reforçam a narrativa, não distraem dela.

### Color Philosophy
- **Primária**: Azul profundo (`#1e40af` ou similar) - transmite confiança, ciência, calma mental.
- **Secundária**: Branco puro (`#ffffff`) - clareza, limpeza, esperança.
- **Acentos**: Verde suave (`#10b981` ou similar) - crescimento, melhora, vitalidade.
- **Neutra**: Cinza claro (`#f3f4f6`) - fundo neutro que não compete com o conteúdo.
- **Emoção**: Laranja quente (`#f97316`) - CTA, urgência, ação.

### Layout Paradigm
- **Hero Section**: Assimétrico com texto à esquerda (narrativa) e elemento visual à direita (cérebro/neurônio estilizado).
- **Seções Alternadas**: Texto à esquerda, visual à direita; depois inverte. Cria ritmo visual.
- **Espaçamento Generoso**: Muito ar branco entre seções para respiração visual.
- **Grid Flexível**: Usa 2-3 colunas em desktop, 1 em mobile. Nada centralizado.

### Signature Elements
1. **Neurônio Estilizado**: Um ícone minimalista de um neurônio que aparece em diferentes contextos (hero, seções de benefícios, CTA).
2. **Linhas Conectivas**: Linhas finas que conectam elementos, sugerindo "conexão neural" e fluxo de informação.
3. **Gradientes Sutis**: Gradientes muito suaves (azul → roxo) em backgrounds, nunca vibrantes.

### Interaction Philosophy
- **Hover Effects**: Botões ganham um brilho suave, cards levantam levemente.
- **Scroll Reveals**: Elementos entram suavemente conforme o usuário scroll (fade-in + slide-up).
- **CTA Magnetismo**: O botão final ("Ir para Synaptigen") tem um efeito de "pulsação" suave para atrair atenção.

### Animation
- **Entrance**: Elementos entram com fade-in + translateY(-20px) em 600ms com ease-out.
- **Hover**: Botões escalam 1.05 em 200ms, cards levantam 4px em 300ms.
- **Scroll Trigger**: Elementos se revelam conforme entram no viewport (usando Intersection Observer).
- **CTA Pulse**: O botão final pulsa levemente (opacity 0.8 → 1) a cada 2 segundos.

### Typography System
- **Display**: Poppins Bold (700) para headlines - moderno, confiante.
- **Body**: Inter Regular (400) para corpo de texto - legível, neutro.
- **Accent**: Poppins SemiBold (600) para destaques dentro do texto.
- **Hierarchy**: H1 (48px) → H2 (32px) → H3 (24px) → Body (16px).

---

## Decisão Final
Escolhi o **Neuro-Minimalism** porque:
- Reforça a autoridade científica do Synaptigen sem ser árido.
- O design assimétrico mantém a atenção engajada.
- As cores azul + verde + laranja criam uma progressão emocional clara (confiança → esperança → ação).
- Minimalismo deixa o copywriting brilhar (que é o ativo principal).
